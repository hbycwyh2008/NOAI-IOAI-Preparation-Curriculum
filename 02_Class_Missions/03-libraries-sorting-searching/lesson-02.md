# Mission 03.2 — Search and Sorting

## Required Resource
Teacher-selected CS50 search/sort excerpt.

## Learning Targets
- compare linear and binary search;
- state the precondition for binary search;
- trace bubble and selection sort;
- implement one search and one simple sort.

## Guided Practice
Trace algorithms on paper and count comparisons.

## Independent Task
Implement and test linear search plus selection sort without built-in search/sort shortcuts.
