# Mission 02.2 — Loops and Core Data Structures

## Required Resource
CS50P: Loops.

## Learning Targets
- trace `for` and `while` loops;
- choose an appropriate collection;
- update and query strings, lists, dictionaries, and tuples.

## Guided Practice
Complete collection-manipulation and loop-tracing questions.

## Independent Task
Build a frequency-summary program from a fresh prompt.

## Exit Evidence
Code, two traced examples, one documented bug, and an oral explanation.
