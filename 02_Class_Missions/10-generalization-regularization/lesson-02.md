# Mission 10.2 — Underfitting, Overfitting, and Regularisation

## Required Resource
Google ML Crash Course: Overfitting and Regularization.

## Learning Targets
- diagnose underfitting and overfitting from train/validation errors;
- explain model complexity and data quantity;
- distinguish L1 and L2 regularisation;
- propose a justified remedy.

## Independent Task
Modify one model to reduce overfitting and submit before/after evidence.
