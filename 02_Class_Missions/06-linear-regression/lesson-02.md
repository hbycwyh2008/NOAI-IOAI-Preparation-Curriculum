# Mission 06.2 — Gradient Descent and Multiple Regression

## Required Resource
Andrew Ng MLS Course 1: Gradient Descent and Multiple Features.

## Learning Targets
- explain one gradient-descent update;
- interpret learning rate and convergence;
- explain why feature scaling matters;
- distinguish one-feature and multiple regression.

## Guided Practice
Trace several parameter updates and diagnose two learning curves.

## Independent Task
Compare scaled and unscaled regression training and document the effect.
