# Mission 21.3 — Hugging Face Text Classification

## Required Resource
Hugging Face LLM Course: Sequence Classification.

## Learning Targets
- load a tokenizer and pretrained model;
- apply padding and truncation correctly;
- define labels and a valid metric;
- fine-tune and run inference.

## Independent Task
Fine-tune on a small dataset, inspect errors by class and text length, and save the model card.
