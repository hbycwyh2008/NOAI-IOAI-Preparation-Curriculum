# Mission 01.2 — Input/Output, Types, Conversion, and Debugging

## Required Resource
CS50P: Variables, Types, Exceptions.

## Learning Targets
- predict the type and value of expressions;
- use `input`, conversion functions, and formatted output;
- diagnose common type and value errors.

## Guided Practice
Trace five short programs and repair three broken programs.

## Independent Task
Write a validated input-process-output program without copying the guided example.

## NOAI Check
One type-conversion question, one code trace, and one error diagnosis.
