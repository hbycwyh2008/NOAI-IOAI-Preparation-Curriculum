# Mission 00.2 — Evidence System, GitHub, and Bohrium Readiness

## Learning Targets
- organise work in the required GitHub structure;
- submit evidence with meaningful commits;
- verify Jupyter and Bohrium execution.

## Required Resource
Course setup guides in `01_Student_Start/`.

## Guided Practice
Create the student repository, run one notebook locally or in Bohrium, and commit the output.

## Independent Task
Reproduce the setup from a fresh folder and document one setup failure and fix.

## Exit Evidence
Repository link, environment check, screenshot or log, error note, and commit history.
