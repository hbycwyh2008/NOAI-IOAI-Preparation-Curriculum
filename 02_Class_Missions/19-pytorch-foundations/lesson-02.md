## Required Resource
PyTorch Learn the Basics: Datasets/DataLoaders and Build Model.

## Learning Targets
- explain Dataset versus DataLoader;
- define a small `nn.Module`;
- trace batch and tensor shapes;
- explain what autograd records.

## Independent Task
Build a custom dataset and model from a blank file, then explain the forward method orally.
