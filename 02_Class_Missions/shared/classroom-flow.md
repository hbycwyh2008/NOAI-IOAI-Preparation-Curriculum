# Classroom Flow

| Time | Block | Evidence |
|---|---|---|
| 0–10 | Entry Point Check | individual answers |
| 10–25 | Required Resource | focused notes |
| 25–40 | Talk Round | oral explanation |
| 40–55 | Teacher Diagnosis | corrected misconception |
| 55–70 | Guided Practice | main task |
| 70–80 | Independent Task | rebuild/trace/modify |
| 80–87 | Exit Check | mastery statement |
| 87–90 | Submission | committed evidence |
