# Mission 23.2 — Qwen, Local Inference, and Multimodality

## Required Resource
Current Qwen and Hugging Face official documentation.

## Learning Targets
- load a small model and tokenizer;
- distinguish local inference from API inference;
- explain quantisation and device constraints at a practical level;
- evaluate hallucination, privacy, and prompt-injection risks.

## Independent Task
Run one local or hosted Qwen workflow, log prompts/outputs, and verify claims against a source.
