# Mission 13.2 — Backpropagation and Chain-Rule Tracing

## Required Resource
3Blue1Brown: What Is Backpropagation Really Doing?

## Learning Targets
- explain why gradients are propagated backward;
- trace dependencies in a computational graph;
- calculate one simplified chain-rule gradient;
- distinguish forward values from backward gradients.

## Independent Task
Trace a tiny network from prediction to loss to two parameter gradients.
