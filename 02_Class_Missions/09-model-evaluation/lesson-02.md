# Mission 09.2 — F1, Specificity, ROC/AUC, and Metric Selection

## Required Resource
Google ML Crash Course: Classification Metrics.

## Learning Targets
- calculate specificity and F1;
- explain precision–recall trade-offs;
- interpret ROC/AUC at an introductory level;
- select a metric for an imbalanced task.

## Guided Practice
Solve three confusion-matrix scenarios and defend the metric choice.

## Independent Task
Write a metric-selection memo for fraud, disease screening, or spam detection.
