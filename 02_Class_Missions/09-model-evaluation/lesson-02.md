# Mission 09.2 — F1, Specificity, ROC/AUC, and Metric Selection

**Duration:** 90 minutes

## Timeline

| Time | Block | Student output |
|---|---|---|
| 0–8 min | Entry Point Check | Complete three individual questions without notes. |
| 8–20 min | Required resource | Use only the assigned excerpt. |
| 20–32 min | Focused notes | Record terms, process, example, and misconception. |
| 32–44 min | Talk round | Explain the process; partner challenges vague steps. |

| 54–70 min | Guided practice | Complete the lesson’s guided task. |

## Required Resource
Google ML Crash Course: Classification Metrics.

## Learning Targets
- calculate specificity and F1;
- explain precision–recall trade-offs;
- interpret ROC/AUC at an introductory level;
- select a metric for an imbalanced task.

## Guided Practice
Solve three confusion-matrix scenarios and defend the metric choice.

## Independent Task
Write a metric-selection memo for fraud, disease screening, or spam detection.
