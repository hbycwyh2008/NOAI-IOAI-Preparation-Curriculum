# Mission 09.3 — Cross-Validation and Evaluation Design

## Required Resource
StatQuest: Cross Validation.

## Learning Targets
- explain why one train/test split can be unstable;
- distinguish k-fold, stratified k-fold, and leave-one-out;
- detect leakage in evaluation code.

## Guided Practice
Trace fold assignments and calculate the mean validation score.

## Independent Task
Design an evaluation plan for a small imbalanced dataset and justify every split.
