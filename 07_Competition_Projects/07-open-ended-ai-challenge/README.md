# Open-Ended AI Challenge

## Challenge

Define a baseline, validate it, iterate, and submit under fixed time/compute limits.

## Required Evidence

- data audit;
- validation design;
- first valid baseline;
- experiment table;
- error analysis;
- final model card;
- independent oral explanation.

## Primary Metric

Competition metric

## Constraint

A valid baseline and output file must be produced before advanced experimentation.
