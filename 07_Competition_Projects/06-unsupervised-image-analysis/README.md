# Unsupervised Image Analysis

## Challenge

Solve an image task without reliable labels using clustering, anomaly detection, or structural CV rules.

## Required Evidence

- data audit;
- validation design;
- first valid baseline;
- experiment table;
- error analysis;
- final model card;
- independent oral explanation.

## Primary Metric

Task-specific validation

## Constraint

A valid baseline and output file must be produced before advanced experimentation.
