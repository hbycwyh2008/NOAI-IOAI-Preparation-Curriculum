# NOAI / IOAI Preparation Curriculum

A **competition-first**, **evidence-based**, and **mastery-focused** curriculum for preparing secondary-school students for the NOAI China selection pathway and later IOAI-style open-ended AI challenges.

Students do not receive credit merely for watching videos or running code once. They must **recognize, explain, calculate/trace, rebuild/debug, modify/apply, and compete under constraints**.

## Competition Pathway

```text
Setup
→ Python foundations
→ AI and machine-learning foundations
→ Round 1 paper-based mastery
→ Data analysis and feature engineering
→ Scikit-learn and PyTorch workflows
→ CV / NLP / audio / LLM tasks
→ Round 2 open-ended projects
→ Past-paper reproduction
→ timed mock contests
```

## Evidence Model

Every substantial mission requires:

- focused concept notes;
- a completed practice task;
- an **independent rebuild**;
- an error/debug log;
- a revision record;
- an AI usage note;
- meaningful Git commit history.
