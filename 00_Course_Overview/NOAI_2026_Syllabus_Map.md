# NOAI 2026 Syllabus Map

This map follows the A–D structure of the official 2026 NOAI China academic syllabus (V1.5).

## A. General Computer Skills

- Python syntax, input/output, arithmetic, comparison
- sequence, selection, iteration
- strings, lists, dictionaries, tuples
- functions, modules, packages
- standard modules and third-party libraries
- sorting and searching

## B. AI Foundations

- AI schools of thought, Turing Test, ethics and risk
- supervised, unsupervised, and reinforcement learning
- regression, classification, clustering
- linear and logistic regression
- probability, normal distribution, statistics, distance
- evaluation metrics and cross-validation
- underfitting, overfitting, regularisation
- ensembles, bagging, random forests, gradient boosting
- perceptrons, neural networks, backpropagation, gradient descent
- CNNs, activation/decision/loss functions, Adam/AdamW
- LLM core ideas
- training, validation, and test sets; data cleaning

## C. Advanced Computer Skills

- NumPy, Pandas, Matplotlib
- data cleaning, visualisation, statistical analysis
- feature engineering: windows, pooling, one-hot, moments, embeddings
- forward fill, image augmentation, tokenisation, vocabulary, patching

## D. Advanced AI Skills

- PyTorch tensors, networks, training loops
- CPU/GPU, device movement, mixed precision
- Xavier/He initialisation, batch normalisation, Adam/AdamW
- CNN, RNN, GAN; CV, NLP, audio
- image segmentation, feature extraction, object detection
- LLMs, multimodality, Qwen deployment, API use
