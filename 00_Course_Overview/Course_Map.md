# Course Map

| Phase | Focus | Competition Outcome |
|---|---|---|
| 0 | Setup and workflow | Student can work in GitHub, Jupyter, and Bohrium |
| 1 | Python foundations | Student can read and complete paper-based Python code |
| 2 | AI / ML foundations | Student can explain the required models and concepts |
| 3 | Neural networks | Student can trace forward/backward training logic |
| 4 | Round 1 training | Student can complete a two-hour paper test reliably |
| 5 | Data analysis | Student can clean, transform, and visualise data |
| 6 | Applied ML/DL | Student can build sklearn and PyTorch baselines |
| 7 | Domain tasks | Student can handle CV, NLP, audio, and LLM tasks |
| 8 | Open-ended competition | Student can iterate under time and compute limits |
