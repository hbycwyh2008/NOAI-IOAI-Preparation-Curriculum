# Competition Notebook Rules

A strong competition notebook should run top-to-bottom and contain:

1. imports and reproducibility settings;
2. data loading and inspection;
3. validation strategy;
4. baseline model;
5. metric calculation;
6. error analysis;
7. justified experiments;
8. final training and prediction;
9. submission-file checks.

Avoid uncontrolled notebook state, unnamed variables, unexplained copied cells, and experiments that cannot be reproduced.
