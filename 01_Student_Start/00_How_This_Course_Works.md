# How This Course Works

This is not a video-watching course. Each mission moves through four forms of evidence:

1. **Understand:** focused notes and oral explanation.
2. **Practise:** guided questions or code.
3. **Rebuild:** close the example and reconstruct the method independently.
4. **Transfer:** modify the method or apply it to a new task.

Your teacher may ask you to explain any submitted code orally.
