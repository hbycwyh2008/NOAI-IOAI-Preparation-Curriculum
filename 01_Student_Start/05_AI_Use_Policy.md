# AI Use Policy

AI is permitted as a learning and competition tool only when allowed by the event rules.

## Permitted

- ask for an explanation;
- request a hint;
- debug an error after showing your attempt;
- generate extra practice questions;
- review code you can already explain;
- help interpret documentation.

## Not Permitted

- submit an entire generated solution you cannot explain;
- skip the independent rebuild;
- fabricate experiments or results;
- hide AI use;
- use tools prohibited by the current competition rules.

Every assisted submission requires an AI usage note.
