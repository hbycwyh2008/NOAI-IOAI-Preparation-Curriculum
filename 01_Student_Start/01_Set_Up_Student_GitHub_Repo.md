# Set Up Your Student GitHub Repository

Create a private repository named:

```text
student-noai-preparation
```

Recommended structure:

```text
round-1/python/
round-1/ml-concepts/
round-1/neural-networks/
round-1/mock-tests/
round-2/data-analysis/
round-2/sklearn/
round-2/pytorch/
round-2/computer-vision/
round-2/nlp/
round-2/audio/
round-2/llm/
projects/
experiment-reports/
independent-rebuilds/
error-log/
reflections/
ai-usage-notes/
```

Use meaningful commits such as `add logistic regression rebuild` rather than `update files`.
