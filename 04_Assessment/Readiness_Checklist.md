# Readiness Checklist

## Round 1

- [ ] Finishes two-hour mock with review time.
- [ ] Calculates all core metrics without prompts.
- [ ] Explains BP, CNN flow, regularisation, and ensembles precisely.
- [ ] Reads and completes Python/sklearn code reliably.

## Round 2

- [ ] Creates a valid baseline in 45 minutes or less.
- [ ] Uses train/validation/test data correctly.
- [ ] Builds and debugs a PyTorch loop.
- [ ] Produces valid prediction files.
- [ ] Handles at least tabular, image, text, and audio baselines.
- [ ] Maintains an experiment log under time pressure.
