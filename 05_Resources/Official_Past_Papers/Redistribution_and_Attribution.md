# Official Past-Paper Redistribution and Attribution

Do not copy an official task, dataset, solution, handbook, or rules document into this repository merely because it is publicly viewable.

## Inclusion decision

A file may be committed only when one of the following is true:

1. the official repository/document states a licence that permits redistribution;
2. written permission has been obtained;
3. the file is the teacher's original adaptation and does not reproduce protected material beyond permitted quotation/use.

## Current authorised public archives

- IOAI 2025 task repository: https://github.com/IOAI-official/IOAI-2025  
  The repository states a Creative Commons licence; retain attribution and note modifications.
- IOAI 2024 task repository: https://github.com/IOAI-official/IOAI-2024  
  Retain attribution and check the repository licence before copying individual assets.
- NOAI 2025 public repository: https://github.com/IOAIChina/NOAI-2025  
  Check the repository and each linked asset for an explicit licence before redistribution.

Prefer linking to the official source rather than duplicating large datasets.

## Attribution record

For every imported or adapted item, record:

```text
Original title:
Original author/organisation:
Official source URL:
Licence:
Downloaded/accessed date:
Files used:
Changes made:
Curriculum lesson/assessment using it:
```

## Prohibited actions

- removing original attribution or licence files;
- publishing a teacher-only solution before the organiser releases it;
- copying a handbook that has no redistribution permission;
- presenting an adapted task as an official current paper;
- committing confidential or unreleased 2026 tasks;
- using an official test set as a hidden scored mock after students have access to the labels or solutions.

## Annual review

Before each cohort, recheck the official repository licence, current task-release status, and whether links or files have changed. Annual rules and unreleased tasks remain outside the curriculum repository.
